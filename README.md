# Makian.net

یک اپلیکیشن تحت وب ساده و کاربردی برای مدیریت اطلاعات مرغداری‌ها.

## ویژگی‌ها

- ثبت گزارش روزانه
- محاسبه دور فن
- مدیریت چند سالن
- ذخیره خودکار اطلاعات در `localStorage`
- پشتیبان‌گیری و بازیابی اطلاعات از طریق فایل JSON

## نحوه استفاده

1.  فایل `index.html` را در مرورگر خود باز کنید.
2.  ثبت نام کرده یا وارد شوید.
3.  از منوهای بالای صفحه برای مدیریت سالن‌ها، ثبت گزارش، محاسبه دورفن و ... استفاده کنید.
4.  برای دریافت فایل پشتیبان، روی **خروجی اطلاعات** کلیک کنید.
5.  برای بازیابی اطلاعات از فایل، روی **ورودی اطلاعات** کلیک کرده و فایل JSON را انتخاب کنید.

## نکات مهم

- تمامی اطلاعات در **LocalStorage** مرورگر ذخیره می‌شود. با پاک کردن حافظه مرورگر، اطلاعات از بین خواهد رفت. برای حفظ اطلاعات، حتماً از قابلیت خروجی استفاده کنید.

## فایل‌ها

- `index.html`: ساختار اصلی
- `style.css`: استایل‌ها
- `script.js`: منطق برنامه
