# داشبورد مرغداری

یک اپلیکیشن تحت وب برای مدیریت اطلاعات مرغداری‌ها. این پروژه با HTML، CSS و JavaScript خالص (بدون فریم‌ورک) ساخته شده است.

## امکانات
- **صفحه ورود و ثبت‌نام**: کاربران می‌توانند ثبت‌نام کرده و وارد شوند. اطلاعات در `localStorage` ذخیره می‌شود.
- **مدیریت سالن‌ها**: افزودن، ویرایش و حذف سالن‌های پرورش با اطلاعات نژاد و تاریخ ورود جوجه.
- **گزارش روزانه**: ثبت تلفات، مصرف دان و میانگین وزن (هر ۷ روز یکبار) به تفکیک سالن.
- **ماشین حساب دور فن**: محاسبه دور پیشنهادی فن بر اساس سن، دما، رطوبت و وزن جوجه.
- **نمایش گزارشات**: مشاهده خلاصه و لیست کامل گزارشات ثبت‌شده برای هر سالن.

## نحوه استفاده
1. فایل `index.html` را در مرورگر باز کنید.
2. اگر برای اولین بار است، ثبت‌نام کنید (شماره تماس به عنوان رمز عبور در نظر گرفته می‌شود).
3. پس از ورود، می‌توانید سالن‌ها را تعریف کرده و گزارشات روزانه را ثبت کنید.
4. از منوهای مختلف برای دسترسی به امکانات استفاده نمایید.

## فایل‌ها
- `index.html` : ساختار اصلی صفحه
- `style.css` : استایل‌های ظاهری
- `script.js` : منطق برنامه و تعاملات

## اجرای محلی
کافیست فایل‌ها را در یک پوشه قرار داده و `index.html` را با یک مرورگر باز کنید. نیازی به نصب وابستگی خاصی نیست.

## توسعه‌دهنده
این پروژه به سفارش یک مرغداری و جهت تمرین برنامه‌نویسی فرانت‌اند ایجاد شده است.
